# CScript

Ett modernt kodspråk som genererar optimerad maskinkod.

## Installation
```bash
pip install git+https://github.com/Natella404/CScript.git
```
## Användning
Skapa en fil med ändelsen .cript och börja koda!

## Exempel
```cript
create message {
    store "Hello, World!"
}
display message
```

## Features
- Naturlig syntax
- Optimerad maskinkodsgenerering
- Enkel installation
